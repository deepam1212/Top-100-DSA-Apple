**Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.**

 
```swift
Example 1:

Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
Example 2:

Input: n = 1
Output: ["()"]
```
**Constraints:**

```swift
1 <= n <= 8
```

**Solution**

```swift
class Solution {
    func generateParenthesis(_ n: Int) -> [String] {
        //
        var outputArr: [String] = []
        generate(n, "", 0, 0, &outputArr)
        //
        return outputArr
    }

    func generate(_ n: Int, _ str: String, _ open: Int, _ close: Int, _ output: inout [String]) {
        if str.count == 2 * n {
            output.append(str)
            return
        }
        //
        if open < n {
            generate(n, str + "(", open + 1, close, &output)
        }
        if close < open {
            generate(n, str + ")", open, close + 1, &output)
        }
    }
}
```

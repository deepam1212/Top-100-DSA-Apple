**Given the root of a binary tree, determine if it is a valid binary search tree (BST).**


**A valid BST is defined as follows:**

* **The left subtree of a node contains only nodes with keys strictly less than the node's key.**
* **The right subtree of a node contains only nodes with keys strictly greater than the node's key.**
* **Both the left and right subtrees must also be binary search trees.**


**Example 1:**


![Example](assets/tree1.jpg)


```swift
Input: root = [2,1,3]
Output: true
```

**Example 2:**

![Example](assets/tree2.jpg)

```swift
Input: root = [5,1,4,null,null,3,6]
Output: false
Explanation: The root node's value is 5 but its right child's value is 4.
 ```

**Constraints:**

```swift
The number of nodes in the tree is in the range [1, 104].
-231 <= Node.val <= 231 - 1
```
**Solution**
```swift
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     public var val: Int
 *     public var left: TreeNode?
 *     public var right: TreeNode?
 *     public init() { self.val = 0; self.left = nil; self.right = nil; }
 *     public init(_ val: Int) { self.val = val; self.left = nil; self.right = nil; }
 *     public init(_ val: Int, _ left: TreeNode?, _ right: TreeNode?) {
 *         self.val = val
 *         self.left = left
 *         self.right = right
 *     }
 * }
 */
class Solution {
    func isValidBST(_ root: TreeNode?) -> Bool {
        return validate(root, minVal: Int.min, maxVal: Int.max)
    }

    func validate(_ root: TreeNode?, minVal: Int, maxVal: Int) -> Bool {
        guard let rootNode = root else {return true}
        //
        if rootNode.val >= maxVal || rootNode.val <= minVal {return false}
        //
        return (validate(rootNode.left, minVal: minVal, maxVal: rootNode.val) && validate(rootNode.right, minVal: rootNode.val, maxVal: maxVal))
    }
}
```

**Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0's.**
**You must do it in place.**

**inplace = O(1) Space Complexity**

**Example 1:**
![Trapping Rain Water Example](assets/mat1.jpg)

```swift
Input: matrix = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]
```

**Example 2:**

![Trapping Rain Water Example](assets/mat2.jpg)

```swift
Input: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
Output: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

**Constraints:**
```swift
m == matrix.length
n == matrix[0].length
1 <= m, n <= 200
-231 <= matrix[i][j] <= 231 - 1
```

**Follow up:**
```swift
A straightforward solution using O(mn) space is probably a bad idea.
A simple improvement uses O(m + n) space, but still not the best solution.
Could you devise a constant space solution?
```

**Solution**

```swift
class Solution {
    func setZeroes(_ matrix: inout [[Int]]) {
        //
        let rows: Int = matrix.count
        let coloumns: Int = matrix[0].count
        //
        var isFirstRowZero: Bool = false
        var isFirstColoumnZero: Bool = false
        //
        for j in 0..<coloumns {
            if matrix[0][j] == 0 {
                isFirstRowZero = true
                break
            }
        }
        //
        for i in 0..<rows {
            if matrix[i][0] == 0 {
                isFirstColoumnZero = true
                break
            }
        }
        //
        for i in 1..<rows {
            for j in 1..<coloumns {
                if matrix[i][j] == 0 {
                    matrix[0][j] = 0
                    matrix[i][0] = 0
                }
            }
        }
        //
        for i in 1..<rows {
            for j in 1..<coloumns {
                if matrix[0][j] == 0 || matrix[i][0] == 0 {
                    matrix[i][j] = 0
                }
            }
        }
        //
        if isFirstRowZero {
            for j in 0..<coloumns {
                matrix[0][j] = 0
            }
        }
        //
        if isFirstColoumnZero {
            for i in 0..<rows {
                matrix[i][0] = 0
            }
        }
    }
}
```

**Given an integer array nums of unique elements, return all possible subsets (the power set).**


**The solution set must not contain duplicate subsets. Return the solution in any order.**

 
```swift
Example 1:

Input: nums = [1,2,3]
Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```

```swift
Example 2:

Input: nums = [0]
Output: [[],[0]]
```

**Constraints:**
```swift
1 <= nums.length <= 10
-10 <= nums[i] <= 10
All the numbers of nums are unique.
```
**Solution**

```swift
class Solution {
    func subsets(_ nums: [Int]) -> [[Int]] {
        //
        var answer: [Int] = []
        var result: [[Int]] = []
        backtrack(nums, &answer, &result, 0)
        return result
    }

    func backtrack(_ nums: [Int], _ answer: inout [Int], _ result: inout[[Int]], _ index: Int) {
        if index == nums.count {
            result.append(answer)
            return
        }
        answer.append(nums[index])
        backtrack(nums, &answer, &result, index + 1)

        answer.removeLast()
        backtrack(nums, &answer, &result, index + 1)
    }
}
```

**Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character in t (including duplicates) is included in the window. If there is no such substring, return the empty string "".**


**The testcases will be generated such that the answer is unique.**

 
**Example 1:**
```swift
Input: s = "ADOBECODEBANC", t = "ABC"
Output: "BANC"
```
**Explanation: The minimum window substring "BANC" includes 'A', 'B', and 'C' from string t.**

```swift
Example 2:

Input: s = "a", t = "a"
Output: "a"
```
**Explanation: The entire string s is the minimum window.**
```swift
Example 3:

Input: s = "a", t = "aa"
Output: ""
```
**Explanation: Both 'a's from t must be included in the window.**
**Since the largest window of s only has one 'a', return empty string.**
 

```swift
Constraints:

m == s.length
n == t.length
1 <= m, n <= 105
s and t consist of uppercase and lowercase English letters.
``` 

**Follow up: Could you find an algorithm that runs in O(m + n) time?**

**SOlution**

```swift
class Solution {
    func minWindow(_ s: String, _ t: String) -> String {
        if s.isEmpty || t.isEmpty { return "" }
        
        var need: [Character: Int] = [:]
        for char in t {
            need[char, default: 0] += 1
        }
        
        var window: [Character: Int] = [:]
        let sArray = Array(s)
        
        var left = 0
        var right = 0
        var have = 0
        let needCount = need.count
        
        var minLen = Int.max
        var start = 0
        
        while right < sArray.count {
            let char = sArray[right]
            window[char, default: 0] += 1
            
            if let needVal = need[char], window[char] == needVal {
                have += 1
            }
        
            while have == needCount {
                if (right - left + 1) < minLen {
                    minLen = right - left + 1
                    start = left
                }
                
                let leftChar = sArray[left]
                window[leftChar]! -= 1
                
                if let needVal = need[leftChar],
                   window[leftChar]! < needVal {
                    have -= 1
                }
                
                left += 1
            }
            
            right += 1
        }
        
        return minLen == Int.max ? "" : String(sArray[start..<start + minLen])
    }
}
```

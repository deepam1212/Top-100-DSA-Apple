**Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.**


![Trapping Rain Water Example](assets/rainwatertrap.png)

 
```swift
Example 1:

Input: height = [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6
Explanation: The above elevation map (black section) is represented by array [0,1,0,2,1,0,1,3,2,1,2,1]. In this case, 6 units of rain water (blue section) are being trapped.
Example 2:

Input: height = [4,2,0,3,2,5]
Output: 9
```


**Solution**
```swift
class Solution {
    func trap(_ height: [Int]) -> Int {
        //
        var n: Int = height.count
        //
        var leftMax: [Int] = Array(repeating: 0, count: n)
        var rightMax: [Int] = Array(repeating: 0, count: n)
        //
        leftMax[0] = height[0]
        for index in 1..<n {
            leftMax[index] = max(leftMax[index - 1], height[index])
        }
        //
        rightMax[n - 1] = height[n - 1]
        //
        var myIndex: Int = n - 2
        while(myIndex >= 0) {
            rightMax[myIndex] = max(rightMax[myIndex + 1], height[myIndex])
            myIndex -= 1
        }
        //
        var water: Int = 0
        for i in 0..<n {
            water += max(0, min(leftMax[i], rightMax[i]) - height[i]) 
        }
        //
        return water
   }
}
```
